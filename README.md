## Git Branch Steps
### Finished with your new changes do this
- Step 1
  - Git add .
  - Git commit -m "you commit message"

- Step 2
  - Git checkout master
  - Git pull
  - Git merge branchname -m “Either a dot or your Commit Message”
  - Git push

- Step 3
  - Git checkout branchname
  - Git merge master
